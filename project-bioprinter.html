Handheld Bio-Printer Prototype

Project Overview

This project involved designing a handheld bio-printer prototype for precise bio-ink application, utilizing CAD modeling for component design and 3D printing for fabrication.

Key Features

  • Precise bio-ink application
  • CAD modeling for component design
  • 3D printed fabrication
  • Hands-on assembly and integration

Outcome

The project resulted in a functional and precise bio-printing tool, significantly advancing the lab's research capabilities.
